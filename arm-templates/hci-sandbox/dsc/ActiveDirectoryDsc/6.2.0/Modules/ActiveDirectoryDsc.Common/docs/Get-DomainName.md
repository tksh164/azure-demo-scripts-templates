
# Get-DomainName

## SYNOPSIS
Gets the domain name of this computer.

## SYNTAX

```
Get-DomainName [<CommonParameters>]
```

## DESCRIPTION
The Get-DomainName function is used to get the name of the Active Directory domain that the computer is a
member of.

## EXAMPLES

### EXAMPLE 1
```
Get-DomainName
```
